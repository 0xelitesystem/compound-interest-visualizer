# Compound Interest Visualizer

This tool shows how a starting amount and monthly contributions grow over time with monthly compounding. It returns the ending balance, the total you put in, and the growth on top, and plots the balance year by year with contributions and growth shaded separately.

**Live demo:** https://0xelitesystem.github.io/compound-interest-visualizer/

## What it does

Enter a starting amount, a monthly contribution, an annual rate, and a number of years. The tool compounds monthly: each month earns on the running total, then the contribution is added. It shows the ending balance, total contributed, and growth, and draws the balance curve with the contributed portion and the growth portion stacked.

A fixed rate is an assumption, not a promise. Real returns vary year to year, so the curve is a projection on the inputs you choose, not a forecast.

## Aesthetic

A botanical seed-packet plate: a framed header with a serif title, sage and seed-brown inks, and a growth area chart that reads like a plant climbing the page.

This tool performs arithmetic on the numbers you enter. It is not financial, tax, or accounting advice, and it does not account for your specific circumstances. Verify any figure before acting on it.

## Privacy

Everything runs in your browser. Nothing you type is sent anywhere, stored, or saved. Closing the tab clears it.

## Use it

Open `index.html` in any modern browser, or host it as a static page. No build step, no dependencies, no network calls.
